# daily_app
Eine kleine App welche im Alltag aushilft.

## Aktuelle Features
- Lokale Datenspeicherung, kein Internetzugriff benötigt
- Aufgaben erstellen und als erledigt markieren
- Geldausgaben und Einnahmen hinzufügen und automatisch den Gesamtumsatz berechnen
    - Unterstützt verschiedenste Währungen
- Übersicht aller Features auf einem Startbildschirm
